== Description ==
Add a meta box to blog posts that lets you choose WooCommerce products to relate. Displays those products as a Swiper.js slider at the end of the post content.

== Installation ==
1. Upload the plugin to the `/wp-content/plugins/` directory or use the plugin upload feature in the WordPress admin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit any blog post and select related products from the sidebar.
4. Save and view the post on the front end.
